# init

Installs AI assistant configuration into the current LeadRat repository.

Writes managed files such as `CLAUDE.md`, `CURSOR.md`, `GEMINI.md`, `.github/copilot-instructions.md`, `.cursor/rules/leadrat-ai.mdc`, `.clinerules`, `continue.config.json`, and `roo.md`.

It also installs seed knowledge documents if they do not already exist.

`AGENTS.md` is validated and never overwritten.
